Este projeto se refere ao frontend da aplicação de blog para a matéria de desenvolvimento web II. 

Para iniciar a aplicação, basta executar o comando npm run dev. 

Antes, contudo, é necessário é necessário instalar o módulos necessários. Para isso, é necessário executar o comando npm install.

A aplicação ficará ouvindo na porta 3001.

Utilizou-se no frontend o layout do EJS e EJS-Mate para renderizar as views.
